# Meow Locker

Meow Locker is a Google Chrome extension designed to help you take regular breaks while working or studying.

Set a **save time limit**, and when the timer ends, **Meow** will appear in the center of your screen and block your view, forcing you to stop and rest for the preset **rest time**.

## Features

- Set custom focus time duration
- Set custom rest time duration
- Full-screen Meow popup reminder
- Helps reduce burnout and eye strain
- Simple and lightweight Chrome extension

## How It Works

1. Set your working time.
2. Set your break time.
3. Start focusing.
4. When time is up, Meow appears and blocks the screen.
5. Take your break, then continue again.

## Purpose

Meow Locker helps users build healthier work habits by encouraging regular breaks and reducing nonstop screen time.

## Installation

1. Clone or download this repository.
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked**
5. Select the project folder
